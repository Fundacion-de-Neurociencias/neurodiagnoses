About Neurodiagnoses

Neurodiagnoses is an AI-powered open-source framework designed to advance the diagnosis and prediction of central nervous system (CNS) disorders. The platform leverages cutting-edge machine learning techniques and integrates multi-modal data to generate structured, explainable diagnostic outputs.

Current Prototype: 3-Axis Command-Line Simulator

The latest development is a complete 3-axis diagnostic prototype that runs from the command line, reflecting the project's core tridimensional philosophy:

  • Axis 1 (Etiology): A rules-based engine that assesses genetic risk factors.
  • Axis 2 (Molecular Profile): A machine learning model trained on simulated proteomics data.
  • Axis 3 (Phenotypic Profile): A machine learning model trained on simulated neuroimaging data.

How to Run the Simulation

After setting up the environment, you can run the full simulation with this single command from the repository root:

python run_neurodiagnosis.py
