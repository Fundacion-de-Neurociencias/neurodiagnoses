Reimagining Neurodegenerative Diagnostics

Diseases of the Central Nervous System (CNS) like Alzheimer's or Parkinson's are incredibly complex. Diagnoses often rely on scattered, heterogeneous data—from MRI scans and CSF tests to cognitive assessments and genetic profiles. Neurodiagnoses aims to unify this information into a coherent, probabilistic framework.

Interactive Prototype (Gradio App)

The full 3-axis diagnostic prototype, including the advanced clinical data entry interface, is hosted on Hugging Face Spaces. It allows researchers and collaborators to test the system's capabilities in real-time without needing to run any code.
