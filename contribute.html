Get Involved with Neurodiagnoses

We welcome contributions from AI researchers, neuroscientists, clinicians, and developers. Your involvement can help advance our mission to improve diagnostics and research in neurological diseases.

Ways to Contribute:

  • Report Issues or Feature Requests: Submit these via our GitHub Issues page.
  • Submit Pull Requests: Contribute new functionalities or bug fixes by following our contribution guidelines in the repository.
  • Engage in Discussions: Participate in conversations about methodologies and technical improvements on our GitHub Discussions or join our EBRAINS Forum.
  • Data Contribution: Share datasets by uploading them to our EBRAINS Neurodiagnoses Bucket or the GitHub Data Repository. Ensure all data complies with ethical standards and includes proper documentation.
  • Run AI Training Sessions: Collaborators interested in running their own AI training sessions can:
    1. Clone the repository from GitHub.
    2. Install the required dependencies.
    3. Access our Jupyter Notebooks on EBRAINS for model training and analysis.
    Detailed instructions are available in our GitHub README.

Stay Updated: Follow us on social media and subscribe to our newsletter for the latest updates and opportunities to get involved.
